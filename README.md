Pipeline de Monitoreo Geoespacial: Salud Vegetal en Escuintla, ChiapasEste proyecto implementa un flujo de datos (pipeline) automatizado en Google Cloud Platform (GCP) utilizando la API de Google Earth Engine y Python. El objetivo es monitorear la salud de la vegetación en en un punto de 1 km a la redonda en la región de San Juan Panamá, Escuintla, mediante el cálculo del Índice de Vegetación de Diferencia Normalizada (NDVI).📊 Descripción del ProyectoComo biólogo enfocado en tecnología, desarrollé esta herramienta para automatizar el análisis de cambios en la cobertura forestal. El sistema extrae datos satelitales crudos y los transforma en indicadores procesables.📈 Resultados Obtenidos (San Juan Panamá, Chiapas) | Periodo | Indicador (NDVI) | Estado |
